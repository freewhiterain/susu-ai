# 新人入职指南

> 欢迎加入！本指南帮助你顺利完成入职前三天的所有事项。

---

## 第一天：到达与基础配置

### 上午（09:00 – 12:00）

1. **前台报到**：携带身份证原件 + 录用通知书，前台登记并领取访客牌
2. **HR 见面**：
   - 签署劳动合同（带好银行卡号）
   - 提交入职资料：学历证明、上家离职证明、体检报告
   - 领取员工手册
3. **工牌办理**：HR 带你去行政部拍照，工牌 1 个工作日内制好

### 下午（13:00 – 18:00）

4. **设备领取**（行政部，3 楼）：
   - 笔记本电脑（需签署设备领用单）
   - 鼠标、键盘、电源适配器
   - 门禁卡（押金 50 元，离职归还退款）
5. **账号开通**（IT 部，5 楼，联系 it-support@company.com）：
   - 公司邮箱（firstname.lastname@company.com）
   - 内网 VPN 账号
   - 钉钉企业账号（加入公司组织）
   - 各业务系统账号（由 IT 根据岗位开通）
6. **工位安排**：直属主管带你找到工位，熟悉周围同事

---

## 第二天：系统与工具熟悉

### 必须完成

- [ ] 钉钉安装并绑定企业账号，加入部门群
- [ ] 登录 HR 系统（hr.company.com）完善个人信息
- [ ] 登录 OA 系统（oa.company.com），学习报销/请假流程
- [ ] 配置 VPN，确认能访问内网资源
- [ ] 与直属主管确认试用期 KPI 和第一周工作计划

### IT 常见问题

| 问题 | 解决方式 |
|------|---------|
| 邮箱收不到邮件 | 检查垃圾箱；联系 IT 确认账号激活 |
| VPN 连不上 | 下载最新客户端（内网 IT 页面）；联系 IT 重置密码 |
| 系统权限不足 | 提交 IT 工单，注明所需系统和权限级别 |
| 电脑故障 | 联系 IT（分机 888），当天响应 |

## 第三天：融入与培训

### 新人培训安排（由 HR 统一组织）

| 时间 | 内容 |
|------|------|
| 09:00 – 10:00 | 公司文化与发展历史 |
| 10:00 – 11:00 | 组织架构与各部门职能介绍 |
| 11:00 – 12:00 | 信息安全与保密制度培训（**必须通过考试**）|
| 14:00 – 15:00 | 薪酬福利与绩效制度 |
| 15:00 – 16:00 | 报销、请假、考勤系统操作演示 |
| 16:00 – 17:00 | 与导师/Buddy 见面，制定 30 天成长计划 |

### 第一周额外事项

- 技术岗：完成内网代码仓库权限申请（GitLab 账号）
- 产品/运营岗：完成数据分析平台权限申请
- 销售岗：完成 CRM 系统培训并通过认证

## 常见入职问题

**Q：合同什么时候能拿到副本？**
签完后 HR 会在 3 个工作日内盖章，原件存档，给你留一份扫描件。

**Q：社保和公积金什么时候开始缴？**
入职当月若在 15 号前，当月开始缴；15 号后则次月开始。

**Q：试用期多长？**
P3–P5 岗位：3 个月；P6 及以上：6 个月。表现优秀可申请提前转正。

**Q：钉钉打卡范围是多少？**
公司 500 米范围内均可打卡；出差可申请外勤打卡（需主管确认）。
